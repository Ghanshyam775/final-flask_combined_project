Project Overview

This is the overview of the QR Code Application, which is designed as a final year project. The application includes a QR Code Generator and Scanner, making it simple to create and scan QR codes for various purposes.

How It Works

The application allows users to generate a QR code by entering their data, such as name, email, and custom fields. The generated QR code can be scanned to retrieve this information. It connects with Firebase for storage and authentication.

Features

  • Generate custom QR codes with user data
  • Scan QR codes to retrieve encoded information
  • Firebase authentication and data storage
  • Responsive and user-friendly interface
